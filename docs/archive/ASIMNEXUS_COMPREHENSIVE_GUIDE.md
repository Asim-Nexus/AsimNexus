## Deployment Guide

### Prerequisites

- Python 3.8+
- Docker (for sandboxing)
- NVIDIA GPU (optional, for local LLM)
- OpenAI API key or other LLM provider

### Installation

```bash
# Clone repository
git clone https://github.com/your-repo/asimnexus.git
cd asimnexus

# Install dependencies
pip install -r requirements.txt

# Configure
cp asim_config.json.example asim_config.json
# Edit asim_config.json with your settings
```

### Running ASIMNEXUS

```bash
# Standard mode
python asim.py --mode=standard

# Cloud mode
python asim.py --mode=cloud

# Native mode
python asim.py --mode=native

# Scan mode
python asim.py --mode=scan
```

### Configuration

**File:** `asim_config.json`

```json
{
  "llm_provider": "openai",
  "model": "gpt-4",
  "api_key": "your-api-key",
  "consciousness_level": "personal",
  "life_dimensions": ["personal", "family", "society"],
  "founder_clones_enabled": true,
  "world_systems_enabled": true
}
```

### Docker Deployment

```bash
# Build image
docker build -t asimnexus .

# Run container
docker run -d -p 8000:8000 --gpus all asimnexus
```

## API Reference

### Main Entry Point

**File:** `asim.py`

```python
from asim import ASIMNexus

# Initialize
asim = ASIMNexus()
await asim.initialize()

# Chat
response = await asim.chat("Hello ASIMNEXUS!")

# Get status
status = await asim.get_status()
```

### Core Modules

```python
# MMMM Engine
from core.mmmm_engine import MMMMEngine

result = await asim_triple_brain.think_unified(prompt)

from core.orchestrator import UnifiedOrchestrator
```

## Best Practices

### 1. Use Unified Systems

Always use `world_systems_unified.py` instead of individual system files:

```python
# Good
from core.world_systems_unified import world_systems_manager

# Avoid
from core.healthcare_system import HealthcareSystem
from core.education_system import EducationSystem
```

### 2. Use Orchestrator for Requests

Route all requests through the orchestrator:

```python
# Good
result = await orchestrator.process_request({"type": "status"})

# Avoid
status = await healthcare.get_status()
```

### 3. Use Triple Brain for Complex Decisions

For complex decisions requiring multiple perspectives:

```python
result = await asim_triple_brain.think_unified(
    prompt,
    consciousness_level=ConsciousnessLevel.NATIONAL
)
```

### 4. Follow Security Guidelines

Always use the security framework:

```python
from security.dharma_policy import dharma_policy
from security.audit_log import audit_log

# Check dharma impact
impact = dharma_policy.assess_impact(action)

# Log audit trail
audit_log.log_action(user_id, action, result)
```

## Troubleshooting

### Common Issues

**Issue:** Import errors after reorganization
**Solution:** Update imports to use new locations:
- `orchestrator.py` → `core/orchestrator.py`
- `triple_brain_system.py` → `core/triple_brain_system.py`
- `tool_registry_opencode.py` → `connectors/tool_registry_opencode.py`

**Issue:** CrewAI not available
**Solution:** Install with `pip install crewai langchain-openai`

**Issue:** Langfuse not working
**Solution:** Set credentials in `asim_config.json` or install with `pip install langfuse`
