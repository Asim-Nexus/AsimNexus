## Core Principles

1. **51% Government / 49% Enterprise / 100% Citizen** — Power is distributed to ensure no single entity can dominate
2. **6 Immutable Dharma Rules** — The Veto Engine enforces constitutional boundaries automatically
3. **Time-Bound Agent Contracts** — All AI agents operate under 5/15/30 day contracts with automatic expiry
4. **Offline-First Sovereignty** — Citizens retain full control even without internet connectivity
5. **Transparent Audit Trail** — Every action is logged immutably for accountability
6. **Nepal-First Design** — Built for Nepal's unique digital infrastructure needs
