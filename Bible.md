## 📚 API Endpoints (Actually Working)

### Dharma/ΔT
- `GET /api/dharma/status` - Production ΔT calculations
- `POST /api/dharma/veto` - Manual ethical veto

### Personal OS
- `GET /api/personal/status` - User status
- `GET /api/personal/universe` - 5 layers status
- `GET /api/personal/contracts` - Active contracts

### Agent Mode
- `POST /api/agent/mode/on` - Activate clone
- `POST /api/agent/mode/off` - Deactivate
- `GET /api/agent/status` - Current state

### Mesh Network
- `GET /api/mesh/status` - Mesh status
- `POST /api/mesh/node/init` - Initialize node
- `GET /api/mesh/nodes/discover` - Find peers
- `POST /api/mesh/clone/sync` - Sync clone
- `POST /api/mesh/storage/store` - Distributed storage
- `POST /api/mesh/offline/operation` - CRDT operation

### Sovereignty
- `GET /api/sovereignty/airgap/status` - Air-gap status
- `POST /api/sovereignty/airgap/activate` - Activate air-gap
- `POST /api/sovereignty/airgap/restore` - Restore connection

### System
- `GET /api/system/complete` - Full system status
